# 《傩场·Nuo Field》使用说明

这是一个纯 HTML + CSS + JavaScript 的多页面交互项目，包含开场引导、壁画火把、面具选择、制面、戴面、手诀、法器、傩乐与终章等页面。

## 运行方式

本项目涉及摄像头权限、GLB 模型加载、音频播放和跨页面资源引用，建议通过本地服务器预览，不要直接双击打开 HTML。

公开预览地址是：
https://wowochenzi.github.io/nuo-field/

在本文件夹内运行：

```powershell
node static-server.mjs
```

然后访问：

```text
http://127.0.0.1:4173/
```

如端口被占用，可指定其他端口：

```powershell
$env:PORT=5173
node static-server.mjs
```

然后访问：

```text
http://127.0.0.1:5173/
```

如果使用了自定义端口，请把下方入口中的 `4173` 替换为实际端口。

## 功能预览入口

| 功能 | 推荐入口 | 预览方式与注意事项 |
| --- | --- | --- |
| 完整流程 | `http://127.0.0.1:4173/` | 从总流程入口开始，按页面引导进入开场、壁画和后续模块。 |
| 开场交互 | `http://127.0.0.1:4173/opening.html` | 会加载背景音乐、手势识别资源和开山莽将 GLB；浏览器可能要求用户先点击页面后才允许播放音频。 |
| 壁画火把 | `http://127.0.0.1:4173/pages/mural.html` | 会加载 `assets/mural/torch.glb` 和视频素材；支持鼠标兜底，手势识别需要允许摄像头并能访问 MediaPipe CDN。 |
| 面具 GLB 预览 | `http://127.0.0.1:4173/pages/choose-mask.html` | 会加载 `assets/models/masks/*.glb`；若模型资源加载失败，页面会显示 PNG 兜底图。 |
| 戴面摄像头 | `http://127.0.0.1:4173/pages/wear-mask.html` | 点击页面内的摄像头/模拟按钮后允许浏览器摄像头权限；GLB 面具会叠加到视频画面，识别库不可用时保留基础预览。 |
| 手诀摄像头 | `http://127.0.0.1:4173/pages/gestures.html` | 点击“开启摄像头”并允许权限；用于手势识别和 GLB 面具联动预览。 |
| 傩乐音频 | `http://127.0.0.1:4173/pages/music.html` | 点击乐器播放对应音频；全站右下或页面上的“乐”按钮用于背景音乐开关。 |

## 权限与浏览器要求

- 摄像头功能只在 `localhost` / `127.0.0.1` 或 HTTPS 安全上下文中稳定可用。
- 首次进入摄像头页面时，需要在浏览器权限弹窗中选择“允许”。
- Chrome / Edge 可能会阻止自动播放音频；先点击页面或点击“乐”按钮即可触发播放。
- 戴面、手诀、壁画等识别功能依赖 MediaPipe CDN；离线环境下会降级为鼠标、PNG 或基础 GLB 预览。
- GLB、MP3、MP4、WebM 等资源需要由 `static-server.mjs` 提供正确 MIME 类型，直接打开 HTML 可能导致资源加载失败。

## 主要目录

- `index.html`：总流程入口。
- `opening.html`：开场交互。
- `pages/`：各模块页面。
- `css/`：全局与页面样式。
- `js/`：路由、状态、音频、摄像头和各页面交互逻辑。
- `assets/`：图片、音频、视频、GLB 模型与第三方 Three.js 资源。
